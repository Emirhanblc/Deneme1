Teknik Terimler Sözlüğü

Sistem yönetimi ve backend geliştirme alanında sıkça karşılaşılan teknik terimlerin açıklamaları. Yazılarımda geçen terimleri burada detaylı şekilde açıklıyorum.

Hızlı Navigasyon

A-C D-F G-I J-L M-O P-R S-U V-Z

A-C

API (Application Programming Interface)

Yazılım uygulamalarının birbirleriyle iletişim kurmasını sağlayan arayüz. REST API, GraphQL gibi farklı türleri bulunur. Backend geliştirmede client-server iletişimi için kritik öneme sahiptir.

Backend İlgili Yazılar

CI/CD (Continuous Integration/Continuous Deployment)

Yazılım geliştirme sürecinde kod değişikliklerinin otomatik olarak test edilmesi ve production'a deploy edilmesi süreci. Jenkins, GitLab CI, GitHub Actions gibi araçlarla implemente edilir.

DevOps İlgili Yazılar

D-F

Docker

Uygulamaları container'lar içinde paketleyerek çalıştırmayı sağlayan platform. Uygulamaların farklı ortamlarda tutarlı şekilde çalışmasını garanti eder ve dependency sorunlarını ortadan kaldırır.

Container İlgili Yazılar

G-I

J-L

Kubernetes

Container'ları otomatik olarak deploy etmek, scale etmek ve yönetmek için kullanılan open-source container orchestration platformu. Production ortamlarda container yönetimi için endüstri standardıdır.

Container İlgili Yazılar

LVM (Logical Volume Manager)

Linux'ta fiziksel disk alanını mantıksal birimler halinde yönetmek için kullanılan sistem. Dinamik disk genişletme, snapshot alma ve volume grupları oluşturma gibi advanced özellikler sağlar.

Linux İlgili Yazı

M-O

P-R

Prometheus

Cloud-native uygulamalar için open-source monitoring ve alerting toolkit'i. Time-series veritabanı, multi-dimensional data model ve powerful query language (PromQL) özelliklerine sahiptir.

Monitoring İlgili Yazılar

S-U

SSH (Secure Shell)

Güvenli uzak bağlantı protokolü. Linux sunuculara güvenli erişim sağlar. Public key authentication, port forwarding ve tunneling gibi advanced özellikleri destekler.

Security İlgili Yazı

V-Z

VLAN (Virtual Local Area Network)

Fiziksel network altyapısını mantıksal olarak bölümlere ayıran teknoloji. Broadcast domain'leri ayırarak güvenlik ve performans sağlar. Yedekleme trafiği gibi özel network gereksinimleri için idealdir.

Network İlgili Yazı

VPN (Virtual Private Network)

İnternet üzerinden güvenli ve şifreli bağlantı sağlayan teknoloji. Uzaktan çalışma, güvenli erişim ve coğrafi kısıtlamaları aşmak için kullanılır. OpenVPN, WireGuard gibi implementasyonları bulunur.
