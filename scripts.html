Linux Server Hardening Script

Bash Intermediate

Production Linux sunucuları için otomatik güvenlik hardening script'i. SSH yapılandırması, firewall kuralları, kernel parametreleri ve güvenlik denetimlerini otomatikleştirir.

GitHub Gist'te Görüntüle → İlgili Blog Yazısı → 
#!/bin/bash
# Linux Server Hardening Script
# Production ortamlar için optimize edilmiş

echo "Starting Linux server hardening..."

# SSH hardening
sed -i 's/#Port 22/Port 2222/' /etc/ssh/sshd_config
sed -i 's/PermitRootLogin yes/PermitRootLogin no/' /etc/ssh/sshd_config

# Firewall setup
ufw --force enable
ufw allow 2222

echo "Hardening completed successfully!"

System Health Check Script

Bash Beginner

Linux sunucuların sistem sağlığını kontrol eden basit script. CPU, memory, disk kullanımı ve network durumunu raporlar.

GitHub Gist'te Görüntüle → 
#!/bin/bash
# System Health Check Script

echo "=== System Health Check ==="
echo "CPU Usage: $(top -bn1 | grep "Cpu(s)" | awk '{print $2}')%"
echo "Memory Usage: $(free -h | grep Mem | awk '{print $3"/"$2}')"
echo "Disk Usage:"
df -h | grep -v tmpfs
echo "Uptime: $(uptime -p)"
